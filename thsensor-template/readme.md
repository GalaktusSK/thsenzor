# Project THSensor

Implementation of temperature and humidity smart sensor.


## Hardware Components

* RPi Pico 2 WH
* DHT11/22
* DS3231
* button
* RGB LED/NeoPixel


## Dependencies

_MicroPython_ libs, which need to be downloaded and placed to the `/lib` folder on device.

* [The DS3231 real time clock chip](https://github.com/peterhinch/micropython-samples/tree/master/DS3231) - inexpensive and easily interfaced battery-backed RTC.
* [Microdot](https://microdot.readthedocs.io/en/latest/) - The impossibly small web framework for Python and MicroPython.

Other (optional) dependencies:

* [Pico CSS](https://picocss.com/) - A minimalist and lightweight starter kit that prioritizes semantic syntax, making every HTML element responsive and elegant by default.
